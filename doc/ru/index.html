Настройки

Настройки находятся в файле settings. Для редактирования настроек вы должны использовать любой компилятор для LESS или SASS.

Названия переменной Значение Описание
@color_theme_one #000 Основной цвет
@color_theme_two #fff Основной цвет
@add_color_theme #a74752 Дополнительный цвет. Доступно 11 цветовов. (#a74752, #f1c311, #f4511e, #42a5f5, #26c6da, #825333, #513f66, #9e715a, #1f4b8e, #f5a6b1, #43bd8e).
@add_color_theme2 #eee Дополнительный цвет
@add_color_theme3 #ddd Дополнительный цвет
@color_theme_one_bg rgba(0, 0, 0, 0.7) Цвет для подложки у фоновых изображений.
@font_family 'Raleway', sans-serif Шрифт используемый в теме. Доступно 5 вариантов ('Raleway', sans-serif; 'Roboto', sans-serif; 'Merriweather Sans', sans-serif; 'Dosis', sans-serif; 'Source Sans Pro', sans-serif)

Типографика

В теме используются Google шрифты. Список доступных шрифтов:

Иконки

В теме использую следующие классы для работы с иконками

Название класса Описание
.icon Основной класс. По умолчанию иконки имеют размер 3rem.
.icon_middle Класс для вывода иконок размера 2rem
.icon_rounded Класс для вывода иконок с обводкой
.icon_rounded-hover Класс для эффекта при навдению мыши на иконку
.icon_rounded-black Класс для вывода иконок в черном кружке
.icon_rounded-crimson Классы для вывода в бордовом кружке

<script src="../../js/plugins/jquery.youtubebackground.min.js"></script>

  });

Вспомогательные миксины

Название Описание
.transition(@property, @time, @effect, @delay) Миксин для кросбраузерной реализации свойства transition. @property – свойство для анимации; @time – время анимации; @effect – эффект анимации; @delay – задержка перед началом анимации.
.complex_transition(@property, @property2, @time, @effect, @delay) Миксин для кросбраузерной реализации свойства transition с двумя свойствами. Настройки такие же как у миксина transition.
.translate(@x, @y) Миксин для кроссбраузерной реализации свойства transform с значением translate.
.transform_scale(@x,@y,@value) Миксин для кроссбраузерной реализации свойства transform с значениями translate и scale. @x - значение по оси Х; @y – значение по оси Y; @value – значения для свойства scale.
.absolute_middle_block() Миксин для реализации абсолютного позиционирований по оси Y.
.absolute_center_horizonatal() Миксин для реализации абсолютного позиционирований по оси X.
.absolute_center_block() Миксин для реализации абсолютного позиционирований по осям Х и Y.
.transform_translate_rotateY(@x, @y, @value) Миксин для кроссбраузерной реализации свойства transform с значениями translate и rotateY. @x - значение по оси Х; @y – значение по оси Y; @value – значения для свойства rotateY.
.transform_translate_rotateX(@x, @y, @value) Миксин для кроссбраузерной реализации свойства transform с значениями translate и rotateX. @x - значение по оси Х; @y – значение по оси Y; @value – значения для свойства rotateX.
.transform_translate_rotate3d(@x, @y, @valueX, @valueY, @valueZ, @deg) Миксин для кроссбраузерной реализации свойства transform с значениями translate и rotate3d. @x - значение по оси Х; @y – значение по оси Y; @valueX, @valueY, @valueZ – значения для указания оси, @deg - угол поворота.
.transform_rotate(@value) Миксин для кроссбраузерной реализации свойства transform с значениями rotate. @value – угол поворта.
.transform_simple_scale(@value) Миксин для кроссбраузерной реализации свойства transform с значениями scale. @value – коэффицент для свойства scale.
.transform_origin(@value1, @value2) Миксин для кроссбраузерной реализации свойства transform-origin. @value1, @value2 –значения свойства.
.transform_style(@value) Миксин для кроссбраузерной реализации свойства transform-style. @value - значение свойства.
.clear() Миксин для очистки свойства float.
.bottom_line_decor() Миксин для декоративного нижнего подчеркивания у заголовков и логотипа.
.flexbox() Миксин для кроссбраузерной реализации flexbox.
.flexwrap(@value) Миксин для кроссбраузерной реализации flex-wrap. @value - значение свойства.
.justify_content(@value) Миксин для кроссбраузерной реализации justify-content. @value - значение свойства.
.align_items(@value) Миксин для кроссбраузерной реализации align-items. @value - значение свойства.
.flex_direction(@value) Миксин для кроссбраузерной реализации flex-direction. @value - значение свойства.
